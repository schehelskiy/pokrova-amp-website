**«Покрова АМП»** — це інформаційний ресурс для першої в Україні ампфутбольної команди (футбол для людей з ампутаціями). Проєкт створений для популяризації спорту серед ветеранів та людей з інвалідністю.

## 🌟 Основний функціонал:

### 🏃 Команда та Гравці (Team)
- Повні профілі гравців з біографією та статистикою.
- Розподіл за позиціями та складом команди.

### 📅 Матч-центр (Matches)
- Розклад майбутніх ігор та результати минулих матчів.
- Турнірні таблиці та деталі кожної гри (склади, автори голів).

### 🛒 Інтернет-магазин (Shop)
- Каталог фірмової атрибутики та спортивної екипіровки.
- Кошик товарів та система оформлення замовлень.

### 📰 Новини та Галерея
- Стрічка останніх подій з життя клубу.
- Фотоальбоми з матчів та тренувань.

### 👤 Користувачі
- Реєстрація та авторизація через Django Allauth.
- Особисті кабінети користувачів.

## 🛠 Технологічний стек:

- **Backend:** Django 6.0.1 (Python)
- **Frontend:** HTML5, CSS3, JavaScript (Bootstrap 5)
- **База даних:** SQLite3
- **Керування контентом:** Django Admin (Jazzmin theme), CKEditor для новин.
